# E听说VIP Hook — 编译与部署指南

## 一、环境准备

### 1. Android Studio
- 下载安装 Android Studio (最新版)
- 安装 Android SDK 34

### 2. Xposed API jar
从以下地址下载 `XposedBridgeApi-54.jar`:
- GitHub: https://api.xposed.info/download/XposedBridgeApi-54.jar
- 放置到 `app/libs/` 目录

```bash
mkdir -p app/libs
# 将下载的jar放入 app/libs/XposedBridgeApi-54.jar
```

## 二、编译

### 使用 Android Studio
1. 用 Android Studio 打开本项目目录
2. File → Open → 选择 `ETingshuoVipHook` 文件夹
3. Gradle Sync 等待完成
4. Build → Build APK

### 使用命令行
```bash
cd ETingshuoVipHook
./gradlew assembleRelease
# 输出: app/build/outputs/apk/release/app-release-unsigned.apk
```

### 签名
```bash
# 生成签名密钥(首次)
keytool -genkey -v -keystore release.jks -keyalg RSA -keysize 2048 -validity 10000 -alias hook

# 签名APK
apksigner sign --ks release.jks --out ETingshuoVipHook.apk app-release-unsigned.apk
```

## 三、部署

### 方式A: LSPosed (Root)
1. 安装 `ETingshuoVipHook.apk`
2. 打开LSPosed管理器
3. 模块页 → 启用 "E听说VIP Hook"
4. 点击模块 → 勾选 `com.ets100.secondary`
5. 强制停止E听说中学，重新打开

### 方式B: NPatch (免Root)
1. 安装 NPatch 管理器
2. NPatch → 管理 → 添加应用 → 选择 E听说中学
3. 点击已添加的应用 → 模块 → 选择 "E听说VIP Hook"
4. 点击"修补" → 等待生成修补后的APK
5. 卸载原版E听说中学
6. 安装修补后的APK
7. 打开应用

### 方式C: LSPatch (免Root)
1. 安装 LSPatch 管理器
2. LSPatch → 管理 → 右上角"+" → 选择E听说中学APK
3. 修补模式选择"便携模式"
4. 嵌入模块 → 选择 "E听说VIP Hook"
5. 开始修补 → 安装修补后的APK

## 五、脱壳后精确定制

脱壳完成后，需要用jadx分析真实代码，然后更新以下内容:

### 1. 更新已知类名 (EcardHook.java)
```java
String[] knownModelClasses = {
    "com.ets100.secondary.model.EcardInfo",  // ← 替换为真实类名
    "com.ets100.secondary.bean.VipStatus",    // ← 替换为真实类名
};
```

### 2. 更新VIP状态字段名
在jadx中搜索:
- `str_vip_dub_video_tip1` 的引用 → 找到VIP判断逻辑
- `ic_vip_lock_flag` 的引用 → 找到锁定图标显示逻辑
- `str_expire` / `str_not_expire` → 找到过期判断逻辑
- `BuyEcardWebView` → 找到购买页面入口

### 3. 更新资源ID
如果VIP锁图标资源ID在不同版本有变化，更新 VipStatusHook.java 中的:

if (resId == 0x7f0806bf) {  // ← 更新为当前版本的资源ID
